# 个人待办事项管理系统说明文档

## 一、系统名称

个人待办事项管理系统

## 二、系统简介

本系统面向个人日常学习和生活管理场景，帮助用户记录、查看、修改和完成待办事项。系统采用 C++ 语言实现，当前版本为控制台交互程序，数据通过本地 CSV 文件保存。

## 六、用户手册

### 1. 启动程序

使用 Dev-C++ 打开 `PersonalToDoList.dev`，点击“编译并运行”。

也可以使用命令行编译：

```bash
g++ -std=c++11 -Iinclude src/main.cpp src/task.cpp src/storage.cpp src/ui.cpp -o PersonalToDoList.exe
```

运行：

```bash
PersonalToDoList.exe
```

### 2. 添加待办事项

在主菜单输入 `1`，依次输入：

- 标题
- 描述
- 截止日期
- 优先级

系统会自动生成 ID，并将状态设置为 `Pending`。

### 3. 查看全部待办事项

在主菜单输入 `2`，系统会显示所有待办事项。

### 4. 修改待办事项

在主菜单输入 `3`，输入要修改的待办事项 ID，然后重新输入标题、描述、截止日期和优先级。

### 5. 删除待办事项

在主菜单输入 `4`，输入要删除的待办事项 ID。

### 6. 标记为已完成

在主菜单输入 `5`，输入待办事项 ID，系统会将其状态改为 `Completed`。

### 7. 保存并退出

在主菜单输入 `0`，系统会把当前待办事项保存到 `data/tasks.csv` 并退出程序。
